Staffing Transparency Suite

Connecticut Nursing Home Staffing Transparency Tools

These tools use CMS Payroll-Based Journal staffing data, CMS Provider Information, and CMS SNF Enrollment affiliation context to help users explore nursing home staffing patterns in Connecticut. They are designed for screening, comparison, and question-building, not as stand-alone findings of violations, poor care, or legal responsibility.

Read before using: PBJ staffing is quarterly screening data. CT comparison points are PBJ-derived screening estimates, not formal DPH compliance findings. Case-mix comparison points are contextual CMS Provider Information values. Ownership and affiliation context is informational and does not prove common day-to-day control. Use drill-down tools before drawing conclusions. Read the data methodology.
